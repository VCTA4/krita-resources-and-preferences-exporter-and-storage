preferences (folder): folder in which you can store custom program settings.
installation: move or copy one file named kritarc to the source folder and run the installation file


bakup (folder): the folder in which a backup copy of the previous settings is stored
(in case of a program error after installation, you can restore the previous settings).
This file will be copied to the folder after running the installation file.


Target folder: C:\Users\%USERNAME%\AppData\Local

To reset to default settings, run the file reset_to_default.ini
